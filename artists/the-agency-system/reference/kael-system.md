# The Kael System — Alter Taxonomy

The internal narrative entity of The Agency System. Named after its primary host alter.
Eleven documented alters plus one significant external partner figure.

## System Architecture Notes

### ANPs vs EPs
Per Theory of Structural Dissociation (TSDP):
- **ANPs** (Apparently Normal Parts): Kael, Lex, Alex — handle daily life, logic, external functioning
- **EPs** (Emotional Parts): Nyx, Kiko, Lia, Moros — hold traumatic memory, sensory data, defensive responses
- **Mixed/Other**: Selene (Protector), Rhys (Caretaker), Isabelle (Controller), Argus (Critic)

### The Core Conflict
ANPs want to quarantine trauma to maintain functioning. EPs are perpetually trapped within the trauma. This ANP/EP conflict is the primary narrative friction.

### Integration Philosophy
Not fusion (erasure) but **Functional Multiplicity** — alters retain distinct identities but achieve co-consciousness and cooperation. Every alter's original survival mechanism contains a core competency vital to polyphonic resilience.
