## Genre Combinations

These genres commonly blend well together or have natural crossover potential:

### Rock + Electronic
- [Synthwave](synthwave/README.md) + [Rock](rock/README.md) = Synth-rock, darksynth
- [Industrial](industrial/README.md) + [Metal](metal/README.md) = Industrial metal
- [New Wave](new-wave/README.md) + [Post-Punk](punk/README.md) = Post-punk revival
- [Shoegaze](shoegaze/README.md) + [Electronic](electronic/README.md) = Ethereal wave

### Hip-Hop + Other Genres
- [Hip-Hop](hip-hop/README.md) + [Jazz](jazz/README.md) = Jazz rap, lo-fi hip-hop
- [Trap](trap/README.md) + [R&B](rnb/README.md) = Alternative R&B, trap-soul
- [Hip-Hop](hip-hop/README.md) + [Rock](rock/README.md) = Rap rock, nu-metal
- [Grime](grime/README.md) + [Drill](drill/README.md) = UK drill, hybrid flows

### Folk/Country + Rock
- [Folk](folk/README.md) + [Rock](rock/README.md) = Folk-rock
- [Country](country/README.md) + [Rock](rock/README.md) = Country rock, Americana
- [Bluegrass](bluegrass/README.md) + [Punk](punk/README.md) = Punk bluegrass
- [Celtic Punk](celtic-punk/README.md) = Irish folk + punk

### Electronic Dance Hybrids
- [House](house/README.md) + [Techno](techno/README.md) = Tech house
- [Trance](trance/README.md) + [Progressive Rock](progressive-rock/README.md) = Progressive trance
- [Dubstep](dubstep/README.md) + [Trap](trap/README.md) = Hybrid trap
- [Disco](disco/README.md) + [House](house/README.md) = Nu-disco, French touch

### R&B/Soul Fusions
- [R&B](rnb/README.md) + [Hip-Hop](hip-hop/README.md) = Contemporary R&B
- [Gospel](gospel/README.md) + [R&B](rnb/README.md) = Neo-soul
- [Funk](funk/README.md) + [Rock](rock/README.md) = Funk rock
- [R&B](rnb/README.md) + [Electronic](electronic/README.md) = Alternative R&B

### World/Regional Fusions
- [Latin](latin/README.md) + [Hip-Hop](hip-hop/README.md) = Latin trap, reggaeton
- [Reggae](reggae/README.md) + [Electronic](electronic/README.md) = Dub, dubstep (origins)
- [Afrobeats](afrobeats/README.md) + [Pop](pop/README.md) = Afropop
- [Bossa Nova](bossa-nova/README.md) + [Jazz](jazz/README.md) = Brazilian jazz

### Middle Eastern Fusions
- [Middle Eastern Pop](middle-eastern-pop/README.md) + [Electronic](electronic/README.md) = Arabic-electronic fusion, Acid Arab
- [Middle Eastern Pop](middle-eastern-pop/README.md) + [Hip-Hop](hip-hop/README.md) = Arabic rap, Mizrahi hip-hop
- [Middle Eastern Pop](middle-eastern-pop/README.md) + [Rock](rock/README.md) = Arabic rock, raï-rock (Rachid Taha)
