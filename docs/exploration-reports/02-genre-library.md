# Genre Library Catalog

## Overview

This catalog documents **71 genre definitions** available in the `/genres/` directory. Each genre is fully documented with production characteristics, Suno prompt guidance, subgenres, notable artists, and reference tracks.

**Structure:** All genres follow a consistent template format:
- **Genre Overview** - Historical context and cultural significance
- **Characteristics** - Production, instrumentation, rhythm, vocals, harmony
- **Lyric Conventions** - Rhyme schemes, verse structure, key rules
- **Subgenres & Styles** - Variants with descriptions and BPM ranges
- **Artists** - Notable performers with discographies
- **Suno Prompt Keywords** - Organized by category (Core, Production, Vocal, Energy, etc.)
- **Reference Tracks** - Canonical examples with analysis

## Genre File Structure

### Template Layout

Each genre directory contains a single `README.md` file following this consistent structure:

```
# [Genre Name]

## Genre Overview
- Historical origins and evolution
- Cultural/geographical significance
- Key artists and breakthrough moments
- Position in broader music ecosystem

## Characteristics
- **Production & Instrumentation**: Specific instruments, equipment, effects
- **Rhythmic Characteristics**: BPM range, tempo feel, signature patterns
- **Vocal Approach**: Delivery style, language, processing
- **Structural Elements**: Song structure, hook placement, length
- **Emotional Range**: Typical themes and moods
- **Harmony/Theory**: Chord progressions, melodic approach

## Lyric Conventions
- Default rhyme scheme (e.g., AABB, AAB, verse-chorus)
- Rhyme quality expectations
- Verse structure specifics
- Key rules for the genre
- What to avoid
- Density/pacing guidance for Suno (lines per verse, BPM considerations, topics per verse)

## Subgenres & Styles
[Table with columns: Style | Description | BPM Range | Reference Artists]
- Typically 7-15 subgenres per parent genre
- Each with distinct characteristics, BPM, and artist examples

## Artists
[Table with columns: Artist | Era | Style Focus | Key Albums | Notes]
- 10-20+ notable artists across history and contemporary
- Includes discography and cultural significance

## Suno Prompt Keywords
- **Core Terms** — Genre identifiers, foundational concepts
- **Production & Instrumentation** — Specific sounds, equipment, effects
- **Vocal & Delivery** — Singing style, effects, techniques
- **Energy & Mood** — Emotional qualities, themes
- **Subgenre Specific** — Related styles and fusions
- **Tempo & Feel** — BPM suggestions, rhythmic qualities
- [Additional categories as needed for the genre]

All keywords are comma-separated within code blocks for easy copy-paste into Suno prompts.

## Reference Tracks
- 10-15 canonical tracks per genre
- Each with: Artist - "Track" — brief description covering:
  - Why it's canonical
  - Technical/production elements
  - Cultural/historical significance
  - How it exemplifies genre characteristics

```

## Key Observations

### 1. **BPM Consistency**
- Slow genres (Ambient, Blues, Ballads): 40-100 BPM
- Mid-tempo genres (Rock, Pop, Funk): 90-130 BPM
- Fast genres (EDM, Drum & Bass, Grime, Punk): 140-180 BPM
- Variable genres: Jazz, Classical, Experimental

### 2. **Lyric Convention Patterns**
Most genres follow one of these lyric patterns:
- **Verse-Chorus**: Pop, Rock, Country, R&B, Hip-Hop
- **AAB Blues Form**: Blues, some Soul
- **Call-and-Response**: Afrobeats, Gospel, Funk
- **Narrative/Story**: Country, Folk, Singer-Songwriter
- **Free Form**: Experimental, Avant-Garde genres
- **Call-and-Response with Hook**: Reggae, Dancehall
- **Verse-Chorus-Bridge**: Standard Pop/Rock variants

### 3. **Suno Prompt Organization**
Every genre provides 5-7 keyword categories:
1. **Core Terms** — Always includes main genre name(s) + foundational concepts
2. **Production & Instrumentation** — Specific equipment, effects, tone
3. **Vocal & Delivery** — Vocal style, processing, technique
4. **Energy & Mood** — Emotional qualities, party/intimate/dark/uplifting
5. **Subgenre Specific** — Related genres and fusions
6. **Tempo & Feel** — BPM guidance, rhythmic qualities
7. _(Optional)_ — Additional category if needed (e.g., Orchestration, Sampling, etc.)

### 4. **Subgenre Depth**
- **Broad parent genres** (Rock, Metal, Electronic, Hip-Hop, Pop): 15-25+ subgenres
- **Medium genres** (Country, R&B, Jazz): 8-15 subgenres
- **Niche genres** (Vaporwave, Shoegaze, Phonk): 5-10 subgenres
- **Specialized genres** (Classical, Opera, Children's): 4-8 subgenres

### 5. **Artist Documentation**
- **Historical perspective**: Genres include pioneers, golden-era artists, and modern practitioners
- **Discography**: Key albums and release years documented
- **Cultural context**: Artist roles in genre evolution noted
- **Typical count**: 10-20 artists per genre, with emphasis on canonical/influential figures

### 6. **Reference Tracks**
- **Always included**: 10-15 essential tracks per genre
- **Selection criteria**:
  - Canonical, historically important
  - Demonstrate core genre characteristics
  - Span eras (classic + modern)
  - Show production/arrangement innovation
  - Include artist rationale

### Step 3: Research Requirements

**For Genre Overview:**
- At least 2-3 paragraphs covering historical origin, key era(s), current landscape
- Include 3-5 foundational/iconic artists by name
- Note any cultural or geographic significance
- Explain how it relates to broader music landscape

**For Characteristics:**
- Minimum 5-6 subsections (Production, Rhythm, Vocals, Structure, Emotion, Harmony)
- Include specific BPM range with context
- Name 5+ specific instruments/tools commonly used
- List 3-5 typical chord progressions or theoretical features
- Describe vocal delivery with 3+ specific techniques

**For Subgenres:**
- Minimum 7-8 distinct subgenres
- Each with 1-2 sentence description
- BPM range for each
- 3-4 reference artists per subgenre

**For Artists:**
- 10-20+ artists spanning different eras and styles
- Include birth/active era
- List 2-3 key albums with years
- Add 1-2 sentence cultural/technical note

**For Suno Keywords:**
- 6-7 keyword categories (Core, Production, Vocal, Energy, Subgenre, Tempo, optional)
- 8-12 keywords per category
- Mix specific terms (instrument names, techniques) with broader descriptors

**For Reference Tracks:**
- 10-15 canonical tracks minimum
- Mix classic/foundational with contemporary
- Each with Artist - "Track" title and 1-2 sentence analysis
- Explain why each track is canonical

### Step 4: Check Consistency

Before committing:
1. ✅ Verify BPM range is present and realistic
2. ✅ Ensure lyric density guidance is specific (e.g., "4-5 lines/verse")
3. ✅ Check that Suno keywords are comma-separated (no leading spaces)
4. ✅ Verify all markdown tables have proper pipe formatting
5. ✅ Ensure reference tracks include era mix (old + new)
6. ✅ Validate that subgenre BPM ranges fall within parent genre guidance

### Step 5: Update INDEX.md

Add the new genre to the appropriate category in `/genres/INDEX.md`:
1. Find the category section (e.g., "Rock and Related", "Electronic and Dance")
2. Add table row: `| **[Genre Name]** | Brief description | [README](path/to/genre/README.md) |`
3. Maintain alphabetical order within category
4. Update "Total" count in opening summary

### Step 6: (Optional) Add Artist Subdirectory

For complex genres with many notable artists:
```bash
mkdir -p genres/[new-genre-name]/artists
```
Then create `artists/[artist-name]/README.md` for selected deep-dive artists with:
- Extended biography
- Discography
- Key innovations/contributions
- Influence on genre

## Integration with Workflow

### When to Use This Catalog

1. **Album Planning** — Select genre(s) for new album project
2. **Track Definition** — Understand BPM/structure expectations for specific track
3. **Lyric Writing** — Reference rhyme scheme and density guidance
4. **Suno Prompting** — Copy keywords for generation
5. **Research** — Learn canonical tracks and notable artists
6. **Subgenre Exploration** — Find related genres and variants
7. **Artist Comparison** — Understand how your sound relates to reference artists

### Connection to Skills

This genre library supports the following Claude Skills:

- **Album Conceptualizer** — Uses genre definitions for Phase 2 (Genre & Sonic Direction)
- **Lyric Writer** — Applies lyric conventions (rhyme scheme, density, structure)
- **Suno Engineer** — Incorporates Suno keyword suggestions into prompts
- **Mix Engineer** — References production characteristics for mixing decisions
- **Researcher** — Provides artist and track references for historical understanding

### Example Workflow

1. User requests: "Make a new album"
2. `/bitwize-music:album-conceptualizer` starts planning
3. Phase 2 (Sonic Direction): Select genre from this catalog
4. Genre README provides:
   - BPM range for album
   - Production reference (mix engineer guidance)
   - Lyric convention (writer guidance)
   - Suno keywords (generation guidance)
   - Reference artists (inspiration + research)
5. Writer uses lyric conventions from catalog
6. Suno Engineer uses keyword suggestions
7. Cycle repeats for each track
